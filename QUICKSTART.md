# QUICKSTART

## Opção sem setup
1. Abra a pasta do projeto no GitHub.
2. Leia o `README.md` da pasta.
3. Explore os scripts em `scripts/`.

## Opção com execução local

### Pré-requisitos
- Python 3.8+
- Git

### Exemplo

```bash
git clone <repo>
cd <repo>\Helio_ML_Producao
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Qual projeto abrir primeiro
- Data Scientist: `Helio_ML_Producao`
- Data Engineer: `Pipeline_Midia_Paga`
- Analytics: `Analises_Operacionais`
- Vendas/Funil: `Analise_Funil_RedBalloon`
