# ICP

Subprojeto de analise de ICP (Ideal Customer Profile) com consolidacao e diagnostico de perfil.

## Objetivo
- Identificar padroes de perfil com maior aderencia.
- Produzir saídas para apoio a estrategia comercial.

## Scripts principais
- `SCRIPTS/analise_icp.py`
- `SCRIPTS/analise_detalhada.py`
- `SCRIPTS/analise_completa.py`
- `SCRIPTS/gerar_excel_processado.py`

## Execucao
```bash
cd 03_Analises_Operacionais/ICP/SCRIPTS
python analise_icp.py
```
