## 🔄 Reativação de Projetos

Para reativar um projeto arquivado:

1. Avaliar se ainda é necessário
2. Verificar se não há duplicidade com projetos ativos
3. Mover de volta para a pasta raiz de Projetos
4. Atualizar documentação do inventário
