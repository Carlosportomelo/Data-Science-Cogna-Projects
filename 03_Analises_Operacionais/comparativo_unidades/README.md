# comparativo_unidades

Subprojeto para comparar desempenho entre unidades e destacar divergencias relevantes.

## Objetivo
- Medir performance relativa por unidade.
- Gerar dashboards e comparativos para acompanhamento.

## Scripts principais
- `analise_divergencia.py`
- `scripts/performance_unidades.py`
- `scripts/gerador_dashboard.py`
- `scripts/gerador_html.py`

## Execucao
```bash
cd 03_Analises_Operacionais/comparativo_unidades
python analise_divergencia.py
```
