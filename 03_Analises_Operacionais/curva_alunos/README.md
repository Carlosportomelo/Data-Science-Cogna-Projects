# curva_alunos

Subprojeto para consolidacao e preparacao de base relacionada a curva de alunos.

## Objetivo
- Organizar base para leitura temporal e acompanhamento de evolucao.
- Padronizar extracao para consumo em dashboards.

## Estrutura
- `SCRIPT/gerador_base_looker.py`: geracao de base para consumo analitico.
- `DATA/` e `Outputs/`: entrada e saida operacional.

## Execucao
```bash
cd 03_Analises_Operacionais/curva_alunos/SCRIPT
python gerador_base_looker.py
```
