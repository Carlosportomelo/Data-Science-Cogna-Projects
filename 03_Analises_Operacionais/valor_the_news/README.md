# valor_the_news

Subprojeto para analise de leads e geracao de relatorio no recorte The News.

## Objetivo
- Avaliar qualidade e comportamento dos leads desse recorte.
- Gerar relatorio padronizado para acompanhamento.

## Scripts
- `SCRIPTS/analisar_leads.py`
- `SCRIPTS/gerar_relatorio.py`

## Execucao
```bash
cd 03_Analises_Operacionais/valor_the_news/SCRIPTS
python analisar_leads.py
python gerar_relatorio.py
```
