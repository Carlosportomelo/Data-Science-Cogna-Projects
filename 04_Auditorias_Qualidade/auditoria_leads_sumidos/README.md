# auditoria_leads_sumidos

Subprojeto de auditoria para identificar perdas, duplicidades e inconsistencias em leads/negocios/matriculas.

## Objetivo
- Rastrear registros sumidos entre bases relacionadas.
- Priorizar impacto por unidade/secretaria.

## Scripts principais
- `analise_leads_sumidos.py`
- `analise_duplicados.py`
- `analise_duplicatas_rigorosa.py`
- `analise_profunda_ids.py`
- `ranking_impactos.py`

## Saidas
Arquivos de analise em CSV/XLSX sao gerados para investigacao interna.
