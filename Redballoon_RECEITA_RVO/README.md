# 07_Redballoon_RECEITA_RVO

Projeto de apoio a analise de receita com foco em RVO, contendo scripts de calculo/ajuste operacional.

## Objetivo
- Gerar apoio analitico para leitura de receita.
- Consolidar regras operacionais em scripts reproduziveis.

## Estrutura
- `Scripts/`: scripts de processamento.
- `Outputs/`: saidas de analise (quando aplicavel).

## Script atual
- `Scripts/Desconto unidade.py`: rotina de apoio para ajuste/calculo por unidade.

## Periodo
Outubro/2025 a Marco/2026.
