# Portfólio de Projetos - Inteligência de Dados e IA

## Projetos

### Helio_ML_Producao
Sistema de machine learning para scoring e priorização de leads.

### Pipeline_Midia_Paga
Pipeline de ETL e análise de performance de mídia paga.

### Analises_Operacionais
Análises de negócio, eficiência de canais, curva de alunos e comparativos.

### Analise_Funil_RedBalloon
Análise de conversão com mapeamento de etapas e cenários.

## Setup rápido

```bash
cd Helio_ML_Producao
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Estrutura padrão

```text
projeto/
  scripts/
  Docs/
  README.md
  requirements.txt
```

## Quick Start

```bash
cd Helio_ML_Producao && python Scripts/main.py
cd Pipeline_Midia_Paga && EXECUTAR_PIPELINE.bat
cd Analises_Operacionais && python scripts/run_all.py
cd Analise_Funil_RedBalloon && EXECUTAR_ANALISE.bat
```
